Synthetic Shadow DOM Test

Test Results:

Open the browser console and check:

  1. Visit http://localhost:3000
  2. Open Developer Tools (F12)
  3. In the Console, run: document.querySelector('demo-app').shadowRoot
  4. If it returns null, synthetic shadow is working ✅
  5. If it returns a ShadowRoot object, native shadow is being used ❌

Additional Tests:

In the console, run:

// Check if synthetic shadow polyfill is loaded
console.log('Synthetic Shadow Loaded:', typeof window.ShadowRoot !== 'undefined');

// Check if global styles penetrate
const appElement = document.querySelector('demo-app');
const innerElements = appElement.querySelectorAll('.slds-section');
console.log('SLDS classes found inside component:', innerElements.length > 0);

Expected Behavior with Synthetic Shadow:

  • ✅ Global SLDS styles should apply to all components
  • ✅ You can query elements inside components directly
  • ✅ CSS from parent penetrates into child components
  • ✅ shadowRoot property returns null
  • ✅ Mimics Salesforce platform behavior
